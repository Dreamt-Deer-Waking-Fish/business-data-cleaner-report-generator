# Upwork Case Study: Sales Data Cleaner and Automated Report Generator

## Portfolio Title

I built a Python tool that cleans messy sales exports and generates a client-ready Excel report with charts.

## Client Scenario

A small business receives sales exports from different systems every month. The files use inconsistent column names, contain duplicate orders, and include missing or invalid values. Before this automation, the owner or operations assistant had to manually clean spreadsheets before preparing a sales report.

## My Solution

I created a Python reporting workflow that imports multiple CSV or Excel files, standardizes the order data, removes duplicate orders, calculates core business metrics, and exports both an Excel workbook and chart images.

## Key Deliverables

- Reusable Python CLI script
- Multi-file CSV/XLSX loader
- Column standardization and data cleanup
- Duplicate order detection
- Summary metrics for revenue, order count, and average order value
- Top product and monthly revenue analysis
- Excel workbook with stakeholder-friendly sheets
- PNG charts for visual reporting
- README, screenshot checklist, and generated sample outputs

## Business Result

This type of workflow can reduce a repetitive monthly reporting task from hours to minutes. It also makes the report more consistent because the same cleaning and calculation rules are applied every time.

## Technologies Used

Python, pandas, openpyxl, matplotlib, PowerShell-friendly CLI workflow.

## How I Would Customize It for a Client

For a real engagement, I would map the client's actual export columns, add validation rules for refunds or taxes, and optionally schedule the report to run automatically each month.
